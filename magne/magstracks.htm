This is a new section giving a list of all known 'official' tracks which feature Magne Furuholmen (excluding a-ha tracks). These include all collaborations including Bridges. More information can be found by clicking the appropriate link to the releases which feature the track. The track lengths are shown in brackets after the track name. All tracks listed below are available on CD except for the tracks on the Bridges album Fakkeltog which can be found only on LP.
