Pål Waaktaar, the Norwegian songwriter and guitarist in the band a-ha, formed Savoy in 1996. This website is now open source which means you can contribute by editing the pages yourself. You just need to set-up a GitHub account, go to https://github.com/SuzDent/a-hadiscography.co.uk, fork the site and you can submit an edit as a pull request! Please note that each edit requires approval before it goes live and please do a pull request to one file at a time, thanks. If you don't feel confident to make the edit yourself, you can submit issues at: https://github.com/SuzDent/a-hadiscography.co.uk/issues.

On 8 February 2014, Paul released his first track (download only) under the name "Waaktaar", the song is called "Manmade Lake".
