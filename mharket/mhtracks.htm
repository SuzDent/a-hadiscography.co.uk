This is a new section giving a list of all known 'official' tracks which feature Morten Harket (excluding a-ha tracks). More information can be found by clicking the appropriate link to the releases which feature the track. The track lengths are shown in brackets after the track name. The final column indicates whether or not the track is available on CD. If the track is not available on CD, the last column shows the format on which the track exists.
