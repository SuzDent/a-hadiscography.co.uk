Morten Harket Tracks

This is a new section giving a list of all known 'official' tracks which feature Morten Harket (excluding a-ha tracks). These include tracks from Morten's solo albums and singles, collaborations including tracks where Morten is featured on backing vocals. More information can be found by clicking the appropriate link to the releases which feature the track. The track lengths are shown in brackets after the track name. The final column indicates whether or not the track is available on CD. If the track is not available on CD, the last column shows the format on which the track exists. Please contact me at suzie at a-hadiscography.co.uk (replace the " at " with @ - written in full like this to help prevent spam) with details of any other official mixes which are missing from the list.
