Contact

I woiuld love to hear from you if you have some new information, better quality images or corrections.

This website is now open source which means you can contribute by editing the pages yourself. You just need to set-up a GitHub account, go to https://github.com/SuzDent/a-hadiscography.co.uk, fork the site and you can submit an edit as a pull request! Please note that each edit requires approval before it goes live and please do a pull request to one file at a time, thanks.

Alternatively, you can contact me suzie.dent at a-hadiscography.co.uk (please contact me BEFORE sending scans, thank you). I will acknowledge all e-mails, so if you don't receive a response after a few days, please re-send the message as your mail could be caught up in the large amount of spam sent to this address, thanks!
